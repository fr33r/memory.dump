## Install Instructions

### `oh-my-zsh`

```zsh
> sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

### `fzf`

```zsh
> brew install fzf
```

### `fd`

```zsh
> brew install fd
```

### `bat`

```zsh
> brew install bat
```

### `eva`
